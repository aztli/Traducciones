Welcome to Ninja-IDE


NINJA-IDE (from: "Ninja Is Not Just Another IDE"), is a cross-platform integrated development environment specially design to build Python Applications.


NINJA-IDE provides tools to simplify the Python-software development and handles all kinds of situations thanks to its rich extensibility.

NINJA-IDE Version 1.0, also called Kunai, include the following features:


  • Multi-Language code editor
  • Code Completion
  • Embedded Python Console
  • Manage Projects properties and files
  • Run File and Projects
  • Configurable GUI
  • And many more...

Shortcuts!

  • New File: Ctrl+N
  • Save: Ctrl+S
  • Save All: Ctrl+Shift+S
  • Open File: Ctrl+O
  • Open Project: Ctrl+P
  • Create Project: Ctrl+J
  • Find: Ctrl+F
  • Find & Replace: Ctrl+H
  • Undo: Ctrl+Z
  • Redo: Ctrl+Y
  • Copy: Ctrl+C
  • Paste: Ctrl+V
  • Cut: Ctrl+X
  • Python Documentation: F1
  • Close Tab: Ctrl+W
  • Change Tab (Next): Ctrl+Tab
  • Change Tab (Previous): Ctrl+Shift+Tab
  • Run Project (for Python and HTML): F6
  • Run File (for Python and HTML): Ctrl+F6
  • Show/Hide Console: F4
  • Show/Hide Editor: F3
  • Show/Hide Properties Panel: F2
  • Show/Hide All (except Editor): F11
  • Split Tabs Horizontaly: F10
  • Split Tabs Verticaly: F9
  • Follow Mode: Ctrl+F10
  • Zoom In (in Editor): Ctrl+WheelUp
  • Zoom Out (in Editor): Ctrl+WheelDown
  • Fade Out (outside Editor): Alt+WheelDown
  • Fade In (outside Editor): Alt+WheelUp
  • Comment/Uncomment: CTRL+D
  • Insert Horizontal Line: CTRL+R
  • Insert Title Comment: CTRL+T
  • Indent More: Tab
  • Indent Less: Shift+Tab
  • Print File: Ctrl+I
  • Reload File: F5
